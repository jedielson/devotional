# Quickstart: Load Bible Texts

## Prerequisites

- Node.js and npm (or yarn/pnpm) installed.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Create Data File**:
    -   Create a file at `public/data/bible-texts.json`.
    -   Populate it with an array of Bible text objects, following the structure in `data-model.md`.

## Running the Application

1.  **Start the development server**:
    ```bash
    npm run dev
    ```
2.  Open your browser to [http://localhost:3000](http://localhost:3000) to see the application.
